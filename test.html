VoltaireAI Test Page

This page demonstrates the VoltaireAI floating button and dialog component.

Features

  • Nested ring floating button design (60px outer, 40px inner)
  • Blue theme (#0066cc)
  • Draggable floating button
  • Click to show/hide dialog
  • Draggable dialog window
  • Chat-style message display
  • User messages on the right (blue)
  • AI messages on the left (gray)
  • Markdown rendering support (via marked.js)
  • Typing indicator during API calls

API Controls

Testing Instructions

  1. Start the backend server: cd backend && python main.py
  2. Open this page in a browser: test.html
  3. Click the floating button in the bottom-right corner
  4. Type a message and press Enter or click Send
  5. Try dragging the floating button and dialog
  6. Test with different backend URLs if needed

Console Commands

Open the browser console (F12) and try:

VoltaireAI.toggle() - Toggle dialog visibility
VoltaireAI.sendMessage("Hello!") - Send a message programmatically
VoltaireAI.setSiteId("my-site") - Set site ID
VoltaireAI.getState() - Get current state
